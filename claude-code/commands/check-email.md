---
description: Check and manage email threads in AgentMail
argument-hint: [inbox-id]
---

Help the user read and manage their email threads.

1. To list threads: use `list_threads` for the specified inbox.
2. To read a thread: use `get_thread` and summarize concisely.
3. To reply: use `reply_to_message` with the thread ID.
4. To check for new mail: list threads and highlight unread messages.
