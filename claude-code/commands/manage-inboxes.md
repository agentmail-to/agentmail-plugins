---
description: Create and manage AgentMail email inboxes
argument-hint: [inbox-name]
---

Help the user create and manage AgentMail inboxes.

1. To create: use `create_inbox`. Accept optional username and domain.
2. To list: use `list_inboxes` and show email addresses and creation dates.
3. To delete: use `delete_inbox`. Confirm with the user first.
4. To get details: use `get_inbox` with the inbox ID.
